### How to use that ?

> 1: Check to have version "^8.0" of PHP!

> 2: Use this pattern in CLI to run server

```txt
php -S [host]:[port] index.php
```

> 3: browse this URL:
```txt
http://[host]:[port]
```

Finally, You can upload files in "public/" folder
